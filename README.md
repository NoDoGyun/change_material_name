# change_material_name

Blender add on

change material's name

<p>ex)</p>
<p>before</p>
object1</br>  
  ㄴmaterial1</br>
  ㄴmaterial2</br>
<p></p>
<p></p>
<p>after</p>
object1</br>  
  ㄴobject1->material1</br>
  ㄴobject1->material2</br>
<p></p>
<p></p>
<p></p>
you must not include '->' in before material's name
